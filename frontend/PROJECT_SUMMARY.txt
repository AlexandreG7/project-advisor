===============================================================================
                          API INTEGRATION
===============================================================================

All API endpoints configured in src/services/api.js:

1. POST /api/recommendations
   Input: Questionnaire answers
   Output: { id, recommendations, libraries }
   Usage: submitQuestionnaire(answers)

2. GET /api/recommendations/{id}
   Output: Full recommendation data
   Usage: getRecommendation(id)

3. GET /api/recommendations/{id}/files
   Output: { files: [...] }
   Usage: getRecommendationFiles(id)

4. GET /api/recommendations/{id}/download
   Output: ZIP file download
   Usage: downloadRecommendation(id)

5. GET /api/technologies
   Output: { technologies: [...] }
   Usage: getTechnologies()

6. POST /api/compare
   Input: { technologies: [...] }
   Output: Comparison data
   Usage: compareStacks(techIds)

7. POST /api/advice-requests
   Input: { name, email, subject, message }
   Usage: submitAdviceRequest(data)
